"Sonata" - это Discord-bot(далее - бот), написанный на Python 3.8, предназначенный для прослушивания музыки. Бот связует в себе Discord.py - API для взаимодейстия с серверами Discord, и Yandex-music API - неофициальное API для поиска и скачивания треков с онлайн-сервиса Яндекс.музыки. Бот предназначен для личного, некоммерческого использования. 
Краткая инструкция:
Префикс - "?" - символ, который необходимо писать перед каждым сообщением боту(своеобразное обращение к боту)
?join - присоединение к голосовому каналу, в котором вы находитесь(вы должны находиться в канале)
?play <название песни> - присоединение к каналу, скачивание трека и его проигрывание в канале. Если бот уже играет, то песня добавится в очередь
?stop - пауза
?continue - снятие с паузы
?skip - пропуск трека
В случае ошибки перезапустите бота
